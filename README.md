# Jump2digital

El dia 25 de novembre del 2022 es va fer un taller a Junp2Digital 2022 per aprendre a analitzar les xarxes socials a una aplicació com per aquest exemple Twitter.

Vam utilitzar la llibreria **tweepy** i basant-nos en un [article](https://towardsdatascience.com/how-to-download-and-visualize-your-twitter-network-f009dbbf107b), podràs trobar tots els passos a seguir per analitzar la teva pròpia xarxa social.

Inclou una base de dades d'un usuari, en aquest cas 'Telecogresca'. Aquest usuari de Twitter pertany a un festival sense ànim de lucre organitzant per estudiants. Podem veure també una classificació per grups socials d'aquesta xarxa. 

### Fitxers:

- Easy Graph.ipynb -> Notebook amb els passos a seguir
- Presentacio power point.pdf -> PDF de la presentació, inclou processat de llenguatge
- networkOfFollowers.csv & networkOfFollowersT.csv -> csv amb els nodes de Telecogresca
- edges.csv -> Edges de la xarxa Telecogresca
- nodes.csv -> Nodes de la xarxa Telecogresca
- graph.gephi -> Graf de Telecogresca amb el programa Gephi
- twitterFollowers.png -> Graf personal fet amb matplotLib

### Llibreries

- networkx                     2.8.8
